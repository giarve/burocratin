Burocratin te ayuda a rellenar el modelo 720 a partir de los informes de Interactivebrokers y Degiro.

Burocratin utiliza la tecnología WebAssembly con lo cual una vez la página realiza la carga inicial toda acción es local y ningún dato viaja por la red.

Para cualquier mejora, duda, sugerencia o error puedes crear un ticket.

El modelo 720 generado se puede presentar si es la primera declaración o si se ha realizado alguna venta y reinvertido el importe.

Limitaciones:
  • Sólo rellena información de acciones, no líquidez del broker.
  • El código de país que usará para Degiro será NL y para interactive brokers IE.
  • Modelo 720: Revisar el código de domiciliación del país, por defecto cogerá el del ISIN, pero esto no siempre es correcto.
  • Modelo 720: Revisar la fecha de primera incorporación si tu primera transacción fue antes del año a declarar.

El autor no se hace responsable del uso resultante de esta aplicación.
